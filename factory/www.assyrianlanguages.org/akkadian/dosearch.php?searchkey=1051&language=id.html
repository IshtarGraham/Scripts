arratu  [ÁŠ : ]  (n. fem.)

[Religion]

une malédiction , un sort (jeté) ; arratu arāru : maudire / jetter un sort (à quelqu'un) ; arratu lā napšuri : une malédiction imparable / ne pouvant pas être levée ; : arratu ilāni : quelqu'un maudit des dieux ;

Cf. arāru (2)

Variantes : erretu (1)
