akītu  [Á.KI.IT : , A.KI.TU : , Á.KI.TUM : ]  (n. f., pl. akiātu, akâte ; )

[Religion]

1) fête de la victoire ; 2) (présent aussi dans nom propre) ; [ūm / isin akīti] : le jour / festival de l'Akitu (fête du Nouvel An -Premier Nissan-) ; [bīt akītu] : bâtiment de la fête de l'Akitu ; [akītu ṣēri] : fête de l'Akitu à la campagne ; [bāb akītu] : porte de l'Akitu ;
