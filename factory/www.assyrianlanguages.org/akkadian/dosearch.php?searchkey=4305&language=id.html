arāḫu  (vb. a/a, stat. aruḫ)

G. être urgent, se dépêcher, +vent. : venir rapidement D. accélérer, envoyer rapidement ; hend. : faire rapidement Štn. faire aller qqn., qqch. très vite

Cf. urruḫiš
