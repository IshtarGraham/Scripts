abātu  (vb. a/u)

G. détruire ; ruiner (économie) D. détruire complètement, dévaster, saccager Dt. être complètement détruit Š. faire détruire N. être détruit ; s'effondrer, s'écrouler

Voir aussi : diāšu, gamāru, ḫalāqu, ḫapû, lapātu, mêsu, qatû

Comparaison avec d'autres langues sémitiques :

  • Proto-Sémitique : *habāt
  • Arabe : habit  هَبِت
