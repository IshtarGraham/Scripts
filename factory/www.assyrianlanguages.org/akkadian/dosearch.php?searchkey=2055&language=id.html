alamittu  ()

[Campagne → Arbres]

1) variété sauvage de dattier ; 2) jardin suspendu de Ninive : système de vis sans fin / vis d'Archimède dispositif servant à faire monter l'eau ;

Cf. gišimmaru, saluppu, kurunnu, libbi gišimmari

Variantes : alamūtu, elamittu, ḫulamētu, ḫulamītu
