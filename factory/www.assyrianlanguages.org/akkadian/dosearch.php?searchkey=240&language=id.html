ṭiābu  [DÙG : , DÙG.GA : ]  (vb. a/i)

G. être bon, en bonne santé ; être amical (envers qqn : +eli, ana) D. rendre bon, améliorer, bonifier ; 2) être devenir utile , servir ;

Cf. ṭiābu, ṭubbātiš, ṭābu

Variantes : ṭâbu
