akālu  [GU7 : ]  (vb. a/u)

G. manger ; paître, brouter ; dévaster, consumer, défigurer, enlaidir ; irriter, faire mal ; absorber ; dépenser ; profiter, avoir l'usufruit ; diminuer en largeur (Mathématiques) Gt. se ravager, se dévaster entre eux Š. nourrir, faire paître ; faire consumer ; faire tremper ; satisfaire, contenter ; stat. : être infecté Št. multiplier entre eux, élever au carré (Mathématiques) ; tulâša akālu : nourrir au sein , allaiter ;

Cf. akkilu

Voir aussi : apālu, eṭāru, šaqālu
